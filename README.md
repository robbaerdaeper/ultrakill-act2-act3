# [DrGBase] ULTRAKILL - Enemies - Act II & Act III

Source of the addon: https://steamcommunity.com/sharedfiles/filedetails/?id=3759896279

Layout:

* `lua/` - all the code (entities, autorun, tools)
* `models/`, `materials/`, `sound/` - compiled assets
* `source/` - QC/SMD model sources, not shipped in the addon build (see the ignore list in `addon.json`)

Dependencies are the same as the workshop version (DrGBase + the base ULTRAKILL packs), full list is in the workshop description.

Found a bug or something off from canon - issues and PRs are welcome.
